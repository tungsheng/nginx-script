# Nginx Install Script [WIP]

Ngnix installation script.

## Getting Started

### Installing

Step 1: clone this repo to your computer or server.

```
$ git clone https://github.com/tungsheng/nginx-script.git
```

Step 2: run installation script

```
$ cd dotfiles
$ ./init.sh
```
